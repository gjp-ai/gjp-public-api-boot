# gjp-open-api-boot

Open access APIs — Spring Boot 3.x REST API for unauthenticated/public endpoints.

## 🚀 Quick Start (Local Development)

1. **Run Application**:
    ```bash
    ./tooling/script/run.sh
    ```
    The API will be available at `http://localhost:8081/api/`.

## 📚 Documentation & Developer Assets

All non-source code assets are consolidated in the **[`tooling/`](./tooling/)** directory:

- [**Documentation**](./tooling/docs/) — Coding standards and guides.
- [**Postman Collection**](./tooling/postman/) — Import into Postman to test endpoints.
- [**Scripts**](./tooling/script/) — Run and coverage scripts.
